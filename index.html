What is my flagship project?

I did an interesting project in the web that allowed me to consolidate my knowledge acquired in computer science:

The famous J2ee and AngularJs framework which I based my e-commerce project on.

This project was an e-commerce where the back-end is developed inJ2ee a java framework, running on a restAPI. to route the urls of my ecommerce I use a javascript framework: angularJS.

It was also used to make requests on my server: to display the products and to buy for the users side. insert products for the merchant registered on the platform. An administrator who manages the correct form of merchant and user information. Obviously for the design I touched HTML, CSS, Javascript.

The technologies that I touched during this project are:

    the winning trio of the web.

  • Front-end:
    • HTML - tag language
    • CSS - to design the web page
    • Javascript - to animate, and make AJAX requests.
    • AngularJS - to route urls on our web application or site, handle errors. Make requests.

    • My e-commerce stack

  • Back-end:
    • SQL, MySQL - The database. this is where the data is listed and called from an application.
    • Java - programming language, executing the code via a virtual console the JVM.
    • J2ee - the java far framework, which allows you to create a web application. I was using the servlet system to manage authentication on my application.
    • The famous RESTapi - jersey - I used this API to retrieve the JSON requests according to the processes: push an information, Read an information, Retrieve an information, ... I manage my routing on it and send an adequate response to the request.
